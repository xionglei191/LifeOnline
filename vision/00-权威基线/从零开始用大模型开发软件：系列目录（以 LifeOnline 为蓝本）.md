# 从零开始用大模型开发软件：系列目录（以 LifeOnline 为蓝本）

## 一、这套系列的定位

这是一套面向**没有软件开发经验的小白**的教程目录。

目标不是把读者训练成某一种具体语言的程序员，而是先帮助读者建立：
- 对软件是什么的整体理解
- 对软件工程如何运作的基本认识
- 对前端 / 后端 / 数据库 / 平台的基础地图
- 对大模型时代如何组织软件开发的实践方法
- 对“如何从一个想法走到一个系统”的整体把握

整套内容以 **LifeOnline** 为蓝本。

也就是说，它既是入门教程目录，也是一套围绕真实项目不断生长的“历史书写框架”。

## 二、建议的全书结构

# 第 1 章：软件到底是什么？

### 核心内容
- 软件（Software）是什么
- 程序（Program）和系统（System）的区别
- 工具（Tool）、产品（Product）、平台（Platform）的区别
- 为什么“页面”不等于“软件”
- 为什么“功能”不等于“系统”

### 与 LifeOnline 的连接
- LifeOnline 为什么不是普通 App
- LifeOnline 为什么更像一个联合认知体工程
